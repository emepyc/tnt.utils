# tnt.utils

Utilities library for [TnT](http://github.com/emepyc/tnt)

## Installation

- npm:

```
npm install tnt.utils
```

- github:

```
git clone https://github.com/emepyc/tnt.utils
cd tnt.utils
npm install
gulp build-browser
```
